# Amazon Regional Sales Analysis Dashboard | Power BI

## Project Overview
This project is an interactive Power BI dashboard developed to analyze Amazon regional sales performance. The dashboard provides insights into sales trends, profit distribution, product performance, and regional business growth to support data-driven decision-making.

## Objectives
- Analyze Amazon sales data across different regions
- Identify top-performing products and categories
- Monitor revenue and profit trends
- Compare regional sales performance
- Generate actionable business insights using visual analytics

## Features
- Interactive dashboard with filters and slicers
- Regional sales performance analysis
- Profit and revenue tracking
- Product category visualization
- KPI indicators for business metrics
- Trend analysis using charts and graphs

## Tools & Technologies Used
- Power BI
- Data Cleaning and Transformation
- Power Query
- DAX (Data Analysis Expressions)
- Data Visualization Techniques

## Dashboard Components
- Total Sales KPI
- Total Profit KPI
- Regional Sales Distribution
- Product Category Analysis
- Monthly Sales Trends
- Top Performing Regions

## Key Insights
- Identified high-performing regions contributing maximum revenue
- Analyzed sales patterns and seasonal trends
- Evaluated product performance across regions
- Improved business understanding through visual reporting

## Skills Demonstrated
- Business Intelligence
- Data Visualization
- Dashboard Development
- Data Modeling
- DAX Calculations
- Data Analysis

## Project Outcome
Developed an efficient and interactive business intelligence dashboard that transforms raw sales data into meaningful insights for better business decision-making.
